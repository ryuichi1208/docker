# docker-compose Template

docker-composeのテンプレートプロジェクトです。

## Container List

- httpd
- lamp
- mysql
- nginx
- php7-apache
- php5-nginx
- postgresql
- rails
- redmine

## Usage

※コンテナのディレクトリに `READ.me` がある場合は、そちらを参照してください。

例)lamp

```bash
// lamp に移動
$ cd lamp

// create & start
$ docker-compose up -d
```
